Decorate Wordpress with hearts via Falling-in-Love.

== Description ==

Decorate your Wordpress site with falling hearts on Valentine's Day.

This plugin wraps @rfreebern's "Falling-in-Love" Javascript to allow easy installation
under Wordpress.

You can find his code at 

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-share-the-love.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
